# Retired: v2-plan-structure-lint

Archived 2026-04-24. The `pre-commit-zz-plan-structure.sh` hook and its inline structural checks were retired because they duplicate scope that the Orianna v2 Opus agent gate explicitly declines to enforce. Orianna v2 operates with a substance-over-format discipline: it does not block on purely structural "tiny bits" such as bare-filename backtick path references, canonical heading format, or forward self-references in plan prose. The mechanical pre-commit lint produced false positives on non-path backtick tokens in plan prose (verified in the 2026-04-23 session), and an Explore audit identified it as the sole hook in the 15-hook chain that was redundant with respect to Orianna. Structural validation of plan files now lives entirely with the Orianna agent gate (see `plans/implemented/personal/2026-04-23-orianna-gate-simplification.md`). The 15 other hooks remain active.
