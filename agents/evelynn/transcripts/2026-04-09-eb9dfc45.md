---

On auto-compact: there's **no setting to disable it**. `cleanupPeriodDays` controls transcript retention, not context compaction. Auto-compaction is a built-in behavior when the context window fills up — Claude Code automatically summarizes older messages to make room. It can't be turned off via settings.

You can use `/compact` manually to control when it happens, and you can use a `PreCompact` hook to inject instructions about what to preserve. But you can't prevent it entirely — it's a fundamental context window management mechanism.
