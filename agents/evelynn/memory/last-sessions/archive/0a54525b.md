## Key realizations

- **GitHub ruleset UI bypass is broken for `pull_request` rule type on personal repos** (community discussion #113172, ≥1y open). `current_user_can_bypass: "always"` via API does not guarantee UI bypass checkbox. - **`actor_type: "User"`** in ruleset bypass_actors silently doesn't work on personal repos; `RepositoryRole` with admin (5) does (via API — but UI still didn't expose bypass for pull_request rule, hence the revert to classic).
