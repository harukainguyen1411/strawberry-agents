## Key learnings surfaced this session (post-3e165c51)

1. BSD awk vs gawk silent failover — gawk-only 3-arg match() syntax errors on BSD awk at compile time, silently promotes fallback path. If a script has try-primary-fallback-sed and primary uses gawk extensions, the fallback is the real path on macOS. Always test both paths explicitly.
2. Greedy sed on @@ hunk headers — `sed 's/.*+\([0-9,]*\).*/\1/'` breaks on trailing context with `+<digits>`. Anchor to `^@@ ` explicitly.
3. Claude Max x20 weekly Opus cap is a ceiling (~24–40 hrs/week). More seats don't help if Opus cap is the binding constraint. Run /status before starting plan-heavy sessions.
4. Prompt caching (cache_control: ephemeral) is the highest-ROI unexercised lever for Claude capacity — 90% discount on cached input tokens.
