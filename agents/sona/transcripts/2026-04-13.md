## Sona — 2026-04-13T03:13:53Z

The Korotovsky server is a community-built Slack MCP server that works by using your **browser session cookies** instead of an API token.

When you're logged into Slack in Chrome/Safari, your browser stores session cookies that identify you. The server uses those cookies to make API calls as if it *is* you browsing Slack — so it has exactly the same access you do.

**The two cookies it needs:**
- `xoxc-...` — your Slack client token
- `xoxd-...` — your Slack session cookie

You'd extract these from Chrome DevTools (one-time, ~2 minutes), paste them into the config, and it just works.

**The catch:** Slack session cookies expire when you log out or after a long period of inactivity. When that happens, you'd need to re-extract them. In practice, if you stay logged in to Slack (which most people do), they last a long time.

---

Alternatively, there's a cleaner middle ground I should mention: you could build a **custom MCP in your `mcps/` folder** that uses a proper Slack User OAuth token (`xoxp-`). That token never expires unless revoked, and it has full user-level access including DMs and search. It would follow the exact same pattern as your other MCPs (bash start script, added to `.mcp.json`).

# /save-transcript — Save Clean Session Transcript

Saves the current session as a readable markdown conversation log. Strips all tool calls, tool results, thinking blocks, and system messages — keeps only what was said.

## Step 1 — Find the latest transcript JSONL

The transcript files live at:

```

ls -t ~/.claude/projects/-Users-duongntd99-Documents-Work-mmp-workspace/*.jsonl | head -1
```

Note the file path for the next step.

## Step 2 — Parse and extract conversation

Run this Python script inline via Bash. Replace `JSONL_PATH` with the path from Step 1:

```bash
python3 - <<'PYEOF'
import json, sys

jsonl_path = "JSONL_PATH"
output_lines = []

with open(jsonl_path) as f:
    for line in f:
        line = line.strip()
        if not line:
            continue
        try:
            record = json.loads(line)
        except json.JSONDecodeError:
            continue

        msg = record.get("message") or record
        role = msg.get("role")
        content = msg.get("content")

        if not role or not content:
            continue

        # Extract text blocks only
        if isinstance(content, list):
            text_parts = [
                block.get("text", "")
                for block in content
                if isinstance(block, dict) and block.get("type") == "text"
            ]
            text = "\n".join(t for t in text_parts if t).strip()
        elif isinstance(content, str):
            text = content.strip()
        else:
            continue

        if not text:
            continue

        if role == "user":
            output_lines.append(f"**Duong:** {text}\n")
        elif role == "assistant":
            output_lines.append(f"**Sona:** {text}\n")

print("\n".join(output_lines))
PYEOF
```
