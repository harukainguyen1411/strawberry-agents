## Step 0 — Context probe

Run these Bash commands in a single call:

```
pwd
date -u +%Y-%m-%dT%H:%M:%SZ
```

## Step 1 — Probe run

Run the cleaner to a throwaway path first:

```
python3 /Users/duongntd99/Documents/Work/mmp/workspace/scripts/clean-transcript.py --agent sona --session auto --out /tmp/save-transcript-probe.md
```

Capture stderr for any `CLEANER: chain ambiguous` warnings. If chain is ambiguous, relay to Duong before proceeding. Delete the probe file after.

## Step 2 — Clean transcript

Run the cleaner for real, writing to the date-named output:

```
python3 /Users/duongntd99/Documents/Work/mmp/workspace/scripts/clean-transcript.py --agent sona --session auto --out /Users/duongntd99/Documents/Work/mmp/workspace/secretary/agents/sona/transcripts/$(date +%Y-%m-%d).md
```

Exit code handling:
- Exit 0: record the output path from stdout.
- Exit 1: report the error and stop.
- Exit 2: report the internal error and stop.
- Exit 3: secret pattern matched — stop immediately, do not stage any files, escalate to Duong.

## Step 3 — Confirm

Report:
- The file path written
- Message counts (user / assistant exchanges)
- Chain length (number of jsonl files stitched together)
- Any warnings from stderr

## Notes

- The cleaner writes a verbatim Markdown transcript with `## Duong — <timestamp>` and `## Sona — <timestamp>` headers for each turn.
- Tool calls, tool results, system reminders, extended thinking, sidechain records, and harness-injected context blocks are stripped.
- Multi-part assistant messages keep only text portions.
- If no qualifying messages survive, the output will note that and the file will still be written.
