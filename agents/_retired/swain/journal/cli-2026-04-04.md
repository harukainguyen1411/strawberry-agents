Designed the full architecture for replacing the four-step contributor pipeline (Discord → Gemini → GitHub → GHA) with a two-step flow (Discord → Claude CLI as Evelynn). Three-layer design:

1. Thin Discord relay bot (discord.js, pure I/O)
