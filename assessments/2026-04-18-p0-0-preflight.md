### Step-by-step

1. Open: https://github.com/apps/firebase-hosting-preview-channels/installations/new
   - If prompted to choose an account, select **harukainguyen1411**

2. Under "Repository access", select **Only select repositories**

3. Add both:
   - `harukainguyen1411/strawberry-app`
   - `harukainguyen1411/strawberry-agents`

4. Click **Install** (or **Save** if already installed for the account)

5. Success looks like: you are redirected back to GitHub with no error, and if you visit
   https://github.com/settings/installations (under harukainguyen1411 account) you see
   "Firebase Hosting" listed with both repos in scope.

### Alternative path (if you already have Firebase Hosting app installed on harukainguyen1411)

1. Go to: https://github.com/settings/installations
2. Find "Firebase Hosting" — click **Configure**
3. Under "Repository access" add the two new repos
4. Save

### What to confirm afterward

- Both repos visible in Firebase Console under your project's Hosting settings
  (Firebase Console > Hosting > Connected to GitHub)
