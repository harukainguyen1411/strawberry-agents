# GCP MCP Server

Wraps `@google-cloud/gcloud-mcp` — official Google Cloud MCP server that provides a natural language interface to all GCP services via the local `gcloud` CLI.

## Prerequisites

- `gcloud` CLI installed (`brew install --cask google-cloud-sdk` or https://cloud.google.com/sdk)
- Authenticated: `gcloud auth login`
- `npx` available on PATH

## Usage

Started automatically by Claude Code via `.mcp.json`. No API key needed — uses the local gcloud credential chain.
