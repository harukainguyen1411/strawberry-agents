# projects/

This directory holds project-context docs for both concerns.

Each project lives in `projects/<concern>/<status>/<slug>.md` and tracks goal,
definition-of-done, constraints, and active decisions for one focused effort.

The doctrine that defines this structure lives in:

  plans/approved/personal/2026-04-25-project-based-context-doctrine.md
